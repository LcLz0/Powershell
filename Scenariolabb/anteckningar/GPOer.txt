* 4) vaktisLocalAdministratorSundsvall

4) networkSettnings
-------------------------------------------------------------
	- Klienter ska använda DHCP

* 5) RestrictedStockholm
-------------------------------------------------------------
	- låst installera/avinstallera program
	- låst control panel
	- låst program: cmd, powershell, regedit
	- låst task manager
	- Remove run
	- Turn on script execution (Disabled)

* 6) UnrestrictedSundsvall
-------------------------------------------------------------
	- öppen installera/avinstallera program
	- öppen control panel
	- öppen program: cmd, powershell, regedit
	- öppen task manager

* 7) Default Domain Group Policy (password)
-------------------------------------------------------------
	- Gemensamma för hela domänen
	- Sätt bra settings för lösenord och dokumentera här

* 8) administratorsLocalFileservers
-------------------------------------------------------------
    - Endast administratörer skall kunna logga in lokalt på filservrarna.

XX 9) desktopImage
